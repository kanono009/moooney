# AutoClicker

A high-precision Auto Clicker Android application.

## Features
- Floating control panel for easy access.
- Precision timing (busy-wait loop for sub-millisecond accuracy).
- Frame-locked second tap for maximum reliability.
- Accessibility service integration.

## Setup
1. Clone the repository.
2. Open in Android Studio.
3. Build and run on an Android device (API 26+).

## CI/CD
This project is pre-configured for **Codemagic**.
- The `codemagic.yaml` file is included in the root directory.
- Simply connect your GitHub repository to Codemagic and start building.
